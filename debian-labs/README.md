# Debian 13.4.0 Trixie

## Objetivo
Este laboratio esta pensado y creado para la administración de sistemas linux; sin embargo es para aprobar la certificación oficial de CompTIA Linux+

## ¿Por qué Debian?
-Es una distribución FLOSS pensada para toda su comunidad Linux (Free Libre Open Sour Software)

-Licencia GPL,para poder aprender,modificarla,distribuirla sin necesidad de pagar por una licencia

-Es estable y muy usada en servdiores

-Es la madre base de muchas distrubuciones de Ubuntu como sus derivados de está misma

-Permite aprender administración pura sin tantas capas gráficas

-Soporte futuro en cuanto a hardware

-Debian es confianza
